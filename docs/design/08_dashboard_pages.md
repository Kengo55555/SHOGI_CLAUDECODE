```typescript
interface AppContext {
  user: User | null;              // ログインユーザー
```

### 8.2 対局画面の状態

```typescript
// useReducer で管理
interface GamePageState {
```

## 9. テスト観点

| テスト項目 | 内容 |
|-----------|------|
| 未ログイン | トップページが表示され、ダッシュボードにアクセスできないこと |
| ログイン後遷移 | ダッシュボードにリダイレクトされること |
| ナビゲーション | 全画面間の遷移が正しく動作すること |
| レスポンシブ | PC/タブレット/スマホでレイアウトが崩れないこと |
| 通知バッジ | 未読数が正しく表示・更新されること |
| 募集一覧 | リアルタイムで新着募集が表示されること |
| 対局開始 | 募集応諾後に対局画面に遷移すること |
